Hello World!

A program to greet the planet.

This page is an atempt to greet everyone on Earth using our various human languages. I'm interested to see how long this list can become.

I'll start with langueages I know without having to use duckduckgo.

  • English: Hello, world!
  • Spanish: Hola, mundo!
  • Japanese: Konnichiwa, sekai!
  • French: Bonjour, monde!
  • Kinyarwanda: Mwirwe, isi!
  • German: Hallo, welt!
